# Gemini Subtitle Translator (media_utils)

이 프로젝트는 Google Gemini API를 활용하여 영어 자막(.srt) 파일을 타임코드 오차 없이 자연스러운 한국어 존댓말 자막으로 번역하고 정제하는 자동화 도구 및 웹 서비스입니다. 

자막 정합성 검증과 후처리를 개별 명령행 스크립트로 작동시킬 수도 있으며, 아름다운 다크 모드 GUI 기반의 웹 서비스를 통합 실행하여 실시간으로 확인하며 일괄 작업을 수행할 수도 있습니다.

---

## 🚀 주요 기능
1. **자막 문장 구분 전처리 (`pre_srt.py`)**
   - 영어 자막의 문장 구분을 자연스럽게 병합하고, 자막 파일 내에 섞여 있는 박수 소리(`[applause]`), 웃음 소리(`[laughter]`) 등의 노이즈 텍스트를 제거하여 번역 품질을 향상시킵니다.
2. **청크 단위 비동기 번역 & 예외 자동 복구**
   - 전체 자막을 100개 단위의 자막 블록(청크)으로 나누어 번역을 요청하여 속도를 향상시킵니다.
   - API 호출 과정에서 **할당량 초과(429)** 또는 **일시적인 서버 부하(503)** 발생 시 백엔드 단에서 대기 후 자동 재시도합니다.
   - 1차 번역본의 타임라인 및 라인 개수 검증이 실패하면 즉시 더 강력한 대체(Fallback) 모델로 전환하여 다시 번역을 실행합니다.
3. **엄격한 1:1 싱크 대조 검증 (`verify_srt.py`)**
   - 전처리된 원본과 번역본의 총 개수, 순서 인덱스, 타임스탬프가 단 1밀리초도 틀어지지 않았는지 확인하는 정밀 검증 스크립트입니다.
4. **자막 추임새 후처리 (`post_srt.py`)**
   - 번역 완료본에 남은 불필요한 번역 흔적(예: "어, ", "음, ", "네, ", ", 여러분.")을 제거하고 매끄러운 한글 회화체로 정리합니다.
5. **글래스모피즘(Glassmorphism) 기반 실시간 웹 UI**
   - 폴더 경로를 입력하여 내부 SRT 파일을 한눈에 스캔하고 제어합니다.
   - 실시간 스트리밍(SSE) 기술로 모달 창 내의 터미널 스타일 로그와 진행바를 통해 실시간 상태를 파악할 수 있으며 작업이 완료되면 데스크톱 알림을 제공합니다.

## 🛠️ 설치 및 준비물

### 1. 의존성 패키지 설치
가상 환경 활성화 후 아래 명령을 통해 필요한 라이브러리를 설치합니다.
```bash
pip install -r requirements.txt
```

### 2. Gemini API 키 설정
프로젝트 루트 경로에 **`.env`** 파일을 만들고 본인의 Gemini API 키를 저장합니다.
```env
GEMINI_API_KEY=AIzaSyYourGeminiApiKeyHere...
```

## 💻 개별 단독 스크립트 사용법

웹 서비스 없이 개별 Python 스크립트들을 터미널에서 단독으로 유연하게 실행할 수 있습니다.

### 1. 자막 전처리 스크립트 (`pre_srt.py`)
지정한 디렉토리 내의 모든 영어 자막 파일(.srt)을 일괄 전처리하여 `*_pre.srt` 파일을 생성합니다.
```bash
python pre_srt.py [로컬_디렉토리_경로]
```

### 2. 1:1 자막 싱크 검증 스크립트 (`verify_srt.py`)
전처리 원본 자막과 번역본 자막의 모든 인덱스 번호 및 타임스탬프 싱크가 100% 일치하는지 비교합니다.
```bash
python verify_srt.py [원본_pre.srt] [번역_kr.srt]
```
*(검증 통과 시 Exit Code `0`, 불일치/에러 발생 시 Exit Code `1`과 에러 내용을 반환합니다.)*

### 3. 자막 후처리 스크립트 (`post_srt.py`)
지정한 디렉토리 내의 모든 자막 파일들의 한국어 추임새 패턴을 일괄 정제하여 `fixed_*.srt` 파일로 생성합니다.
```bash
python post_srt.py [로컬_디렉토리_경로]
```

## 🌐 통합 웹 서비스 실행법

FastAPI 기반의 모던 웹 인터페이스를 구동하여 브라우저에서 편리하게 마우스 클릭만으로 스캔, 번역 모델 설정, 번역, 검증, 후처리, 모니터링을 통합 제어할 수 있습니다.

### 1. 웹 서버 시작
```bash
python web_server.py
```
*(서버 기동 시 백엔드 콘솔 창에도 표준 출력으로 실시간 진행상황 로그가 출력됩니다.)*

### 2. 웹 UI 접속
서버 구동 후 웹 브라우저를 열고 다음 주소로 접속합니다.
- **[http://127.0.0.1:8000](http://127.0.0.1:8000)**

### 3. 작업 진행
- 화면의 경로에 번역하려는 자막 폴더 경로를 넣고 **[폴더 스캔]**을 실행합니다.
- 번역할 파일 행에서 사용할 **기본 모델**(예: `gemini-3.5-flash`)과 **대체 모델**(예: `gemini-3.1-pro-preview`)을 선택하고 **[번역]**을 누르면 번역 파이프라인이 자동 개시됩니다.
